桌宠开发

AI 桌宠 = 人格化 Agent 入口 。

用户关心： 我能干什么 ？ 底层怎么实现 ？ MCP 、Skills 、Workflow 、Memory 都是基础设施。

情绪价值  + 功能价值

# V1 开发者陪伴助手
目标用户 ： 程序员 、产品经理 、学生 、AI 玩家
### 功能：
    1. 工作状态感知 ： 例如 连续工作60 ， 宠物： 主人 ，已经写了一个小时代码了 ，赶紧袭来活动一下吧 。（端杯茶 递到跟前）
    2. 某个时间段 ，中午 12～2 点 ，凌晨 1～6 点 ，宠物 ：你今天工作已经 11 个小时了 ，真的不睡吗？
    3. Agent 能力，用户：帮我总结日报，调用日报 Skills ，总结 ，执行 ...
    4. 帮我解释 / 翻译 ，看看这个报错 。
